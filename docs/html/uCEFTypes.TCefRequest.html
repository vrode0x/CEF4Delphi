Declaration

type TCefRequest = record

Description

Structure used to represent a web request. The functions of this structure may be called on any thread.

NOTE: This struct is allocated DLL-side.

Implemented by ICefRequest.

<see href="https://bitbucket.org/chromiumembedded/cef/src/master/include/capi/cef_request_capi.h">CEF source file: /include/capi/cef_request_capi.h (cef_request_t))
